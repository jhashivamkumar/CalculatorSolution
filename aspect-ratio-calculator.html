Explanation

The Aspect Ratio Calculator helps you resize an image or video while keeping the same proportions so it does not look stretched. An aspect ratio compares width to height, such as 16:9 (landscape), 9:16 (vertical), 1:1 (square), and 4:5 (portrait).

This is useful for creators and editors working with YouTube thumbnails, YouTube videos, Instagram posts, Reels/Shorts, presentations, web banners, and Canva designs. When you change only width or only height, this tool calculates the matching dimension so the output remains proportional.

If you resize without preserving the ratio, the media can become distorted (faces look wider/taller). To avoid that, choose one dimension as fixed, then compute the other using the aspect ratio. If you must fit a different screen size, you typically choose between: Fit (letterbox) — keeps all content visible with bars, or Crop (fill) — fills the screen but trims edges.

Tip: Common creator ratios — 16:9 for YouTube landscape, 9:16 for Shorts/Reels/Stories, 1:1 for square posts, and 4:5 for portrait Instagram feed posts. Use this calculator to get exact matching width/height quickly.

Formula

Given ratio = ARw : ARh

If width is known:
NewHeight = width × (ARh / ARw)

If height is known:
NewWidth  = height × (ARw / ARh)

FAQs

What is aspect ratio?

Aspect ratio is the relationship between width and height (e.g., 16:9). Keeping it the same prevents stretching.

Which aspect ratio is best for YouTube and reels?

16:9 is standard for YouTube videos; 9:16 is used for Shorts/Reels/Stories; 1:1 is common for square posts; 4:5 is popular for portrait feed posts.

Letterbox vs crop: what should I choose?

Letterbox keeps the full image and adds bars. Crop fills the frame but removes edges. Choose based on what content must remain visible.

Why does upscaling reduce quality?

Upscaling increases pixel dimensions without true detail, which can cause blur. Use higher-res source files and good resampling settings.

Do pixels have to be square?

Most modern screens use square pixels, but some older video formats may use non-square pixels (pixel aspect ratio). If in doubt, use square-pixel presets.
