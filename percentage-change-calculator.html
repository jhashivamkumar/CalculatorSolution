FAQs

When is percent change undefined or infinite?

If the old value is 0, percent change is undefined/infinite because you can‚Äôt divide by zero.

Sign meaning?

Positive means increase and negative means decrease.

Can it exceed 100%?

Yes. If the new value is more than double the old value, percent increase is greater than 100%.

Is this the same as percentage points?

No. Percentage points are the absolute change between percentages (e.g., 10% to 12% is +2 points).
